# Lung Disease Detection Project

**Đề tài 8:** Phát hiện bệnh phổi từ ảnh X-quang ngực

**Thành viên:**
- Võ Hoàng Kim Ngân
- Nhan Gia Huy

**Giảng viên hướng dẫn:** Bùi Tiến Đức

### Các lớp bệnh được phân loại
- COVID19
- NORMAL
- PNEUMONIA
- TUBERCULOSIS

### Công nghệ sử dụng
- U-Net Lung Segmentation (pre-trained weights)
- Hybrid Model: MobileNetV2 + ANN sâu
- Focal Loss + Class Weight
- TensorFlow / Keras

**Kết quả:** Test Accuracy ≈ 91.57%
